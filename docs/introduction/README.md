# Introduction

KuberLogic allows you can deploy, manage and scale software using Kubernetes. KuberLogic is meant to be installed on top of the existing Kubernetes cluster, it could launch other services and monitor them and doesnâ€™t manage the Kubernetes cluster itself.

KuberLogic provides a fully managed common open source applications with the following features:

* Automatic provisioning
* Automatic minor updates
* Automatic failover
* Scheduled backups
* Resource usage monitoring
* Ability to scale up and down as needed
* REST API for service management

Currently, we support:

* MySQL
* PostgreSQL

Next steps:

* Redis
* MongoDB
